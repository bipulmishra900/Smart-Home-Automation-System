# Smart Home Automation System

This project enables remote control and monitoring of home appliances using IoT.

## Features
- Control lights, fans, and appliances from mobile
- Real-time status monitoring
- Wifi-based control using ESP32/NodeMCU
- Supports Blynk / Adafruit / Firebase

## Technologies Used
- ESP32 / NodeMCU
- Relay Module
- IoT Cloud Platform
- Arduino IDE

## How It Works
- Microcontroller sends/receives data via WiFi
- User controls devices from smartphone IoT dashboard
